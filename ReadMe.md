# 使用说明

修改lite_config.json中的参数即可

subscription：修改为自己的订阅链接即可

testmode: 多线程进行，2为两个线程

其他自己看

## btw

运行命令：lite.exe --config ./lite_config.json --test ./other/veg.yaml

test.yaml无作用，就是一个简单的占位置的东西
